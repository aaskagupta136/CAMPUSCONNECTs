
CampusConnect — Ambassador Management Platform

CampusConnect is a modern web application designed to streamline Campus Ambassador programs. It provides dual login flows for Organizations (Admins) and Ambassadors (Students), enabling efficient management and gamified engagement.

---

🚀 Features
- Dual Login System: Separate dashboards for organizations and ambassadors.
- Admin Dashboard:
  - Add ambassadors and assign tasks.
  - Track monthly activity, top performers, and total points.
  - Manage leaderboards by individual and college.
- Ambassador Dashboard:
  - View personal points, streaks, and tasks.
  - Submit proof for completed tasks to earn points.
  - Track weekly performance and milestones.
- Notifications & Milestones: Real‑time alerts for deadlines, achievements, and rank progression.
- Modern UI: Gradient backgrounds, responsive layouts, animated transitions, and clean typography.

---

🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Libraries: FontAwesome, Chart.js, Google Fonts  
- Data Layer: JavaScript objects for ambassadors, tasks, and colleges (demo mode)


📖 How to Use
- Organization Login: Click Login as Admin, enter organization details, and access the admin dashboard to manage ambassadors and tasks.  
- Ambassador Login: Click Login as Ambassador, enter personal details, and access the ambassador dashboard to view tasks, submit proof, and track progress
